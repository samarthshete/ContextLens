# Demo Script

## Goal
Show ContextLens as a serious AI infrastructure project, not a PDF chatbot.

## Demo length
2 to 3 minutes

## Flow
1. Open repository and show README
2. Show architecture diagram/docs briefly
3. Upload a small PDF or policy document
4. Ask a question that is slightly tricky
5. Show retrieved chunks
6. Show final answer
7. Show evaluation scores / failure category
8. Compare two configurations if available
9. End with why this matters: debugging RAG failures systematically

## Voiceover outline
"ContextLens is a RAG evaluation and debugging platform. Instead of treating document-grounded LLM systems as black boxes, it captures the full query trace, evaluates retrieval and answer quality, and classifies where failures happen. In this demo, I’ll upload a document, run a query, inspect the retrieved chunks, review the answer, and show how the system identifies whether the failure came from retrieval, context selection, or unsupported generation."

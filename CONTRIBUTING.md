# Contributing

Thanks for your interest in ContextLens.

## Current status
This repository is under active development. The architecture and roadmap are still being stabilized.

## Ways to contribute
- report bugs
- suggest evaluation metrics
- propose benchmark datasets
- suggest API improvements
- improve docs

## Before opening a PR
1. Read `PROJECT.md`
2. Check `ROADMAP.md`
3. Open an issue first for significant changes

## Development principles
- keep changes scoped
- do not introduce new frameworks without strong reason
- maintain traceability and evaluation clarity
